Azul


Azul is a board game that suits every type of player. It is equal parts puzzle, strategy, and art.

Firstly, let's discuss the negatives. While each game is replayable, after a few rounds you tend to establish a similar plan for each time you play. In addition to that, there is a "mostly correct" way to play as you quickly learn if your opponent(s) have figured it out. The game box description gives away a lot of clues on this "correct" way of playing and this can be frustrating at times if you are not in the know.

Now time for the positives. While I mentioned that you develop a plan early on, I still enjoy playing this game over and over. Additionally, it is a great 3 or 4 player game, but still holds its fun when just two players are available. The game board features two versions which helps eliminate the repetitiveness of the game.
